Refreshing my skills in Java with this (for now) simple todo manager with tickets. 

This is an easy application to keep on track of your business and/or private life with these quick tickets. 

I've used Java, Spring Boot, Junit, HTML and CSS for this application to run. I added tests to make sure the application doesn't crash. 

Future features:
* Adding account and login
* Adding styling via CSS
* Short description to tickets
* Set up in a docker container for continuous running

This application will be developed continuously!
